# GoAegis Quick Start

1. Open the repository in VS Code.
2. Install dependencies.
3. Run the API and MCP server.
4. Open the extension workspace.
5. Save `apps/extension/test-fixtures/vulnerable.py`.
6. Watch the diagnostic appear.
7. Open the GoAegis sidebar.
8. Try `GoAegis: Fix Vulnerability`.
9. Try `GoAegis: Review Current File`.
10. If you have a GitHub token, try `GoAegis: Review Current PR`.

## What To Show First

- The diagnostic
- The sidebar score
- The fix action
- The PR review output
- The trend/history chart

## What To Save For Later

- policy engine rules
- extra scanners
- deployment plumbing
- internal telemetry dashboards
